# Support

## Getting help

If you need help using ZELL:

- Open a GitHub Discussion (preferred for usage questions)
- Open a GitHub Issue (for reproducible bugs)

## Before opening an issue

- Check README.md and docs/SELF_HOSTING.md
- Search existing issues for similar reports
- Gather logs and reproduction steps

## What to include

- Environment details (OS, runtime versions)
- Steps to reproduce
- Expected behavior
- Actual behavior
- Logs, screenshots, or API responses

## Security issues

For vulnerabilities, follow SECURITY.md and avoid public disclosure first.
